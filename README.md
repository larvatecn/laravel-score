# laravel-score

Score Extension for Laravel. Laravel 积分

## 环境需求

- PHP >= 7.4

## Installation

```bash
composer require larva/laravel-score -vv
```

## for Laravel

This service provider must be registered.

```php
// config/app.php

'providers' => [
    '...',
    Larva\Score\ScoreServiceProvider::class,
];
```
